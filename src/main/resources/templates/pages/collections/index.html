Importe un archivo Excel y sincronice pagos sobre los creditos segun el ID de cada cuenta.

Importar Excel

Soporta archivos .xls y .xlsx. Se procesa la primera hoja del archivo.

Columnas esperadas: ID credito, Monto/Pago/Cobro y opcional Metodo

Si la planilla no trae columna de metodo, se usa este valor.

Cada fila valida impacta sobre el credito correspondiente.

Formato sugerido

ID credito Monto Metodo
1001 5500 EFECTIVO
1002 3000 TRANSFERENCIA
1003 2500 opcional
  • Si la hoja tiene encabezados, el importador detecta columnas por nombre.
  • Si no hay encabezados, intenta resolver por posicion de ID y monto en cada fila.
  • Si una fila falla, las demas igualmente se procesan.
